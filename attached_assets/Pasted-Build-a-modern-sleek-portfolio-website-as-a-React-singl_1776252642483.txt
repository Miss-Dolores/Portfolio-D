✨ GLOBAL INTERACTIONS & ANIMATIONS
=============================================================

PAGE TRANSITIONS:
- On nav link click → trigger a CSS class that runs:
  opacity: 1→0 + translateY(0→-10px) in 250ms
  then scroll to section
  then reverse: opacity: 0→1 + translateY(10px→0)
- Implement with useRef + className toggling in React

HOVER EFFECTS (all buttons):
- Sliding background fill (::before pseudo via inline style 
  or Tailwind arbitrary)
- Green border glow (box-shadow: 0 0 12px #22c55e)
- Scale: 1 → 1.05
- Duration: 0.3s ease

STAGGERED LOAD (Projects):
- Use IntersectionObserver in a useEffect hook
- Add CSS custom property --delay per card
- Animate: opacity 0→1 + transform translateY(30px→0)

MARQUEE (Skills):
- Pure CSS @keyframes marquee animation
- Two rows scrolling in opposite directions
- Duplicate skill list for seamless loop
- Pause animation on hover (animation-play-state: paused)
